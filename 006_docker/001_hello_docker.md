# Docker

## Docker 是什么？

> Docker 是一款开源的「应用打包 & 环境隔离」工具，基于 Linux 容器技术，可以把 软件 + 依赖 + 运行环境 + 配置 整体打包，实现：
> 一次打包，到处运行

> Docker 是一个标准的集装箱，箱子里有完整的封装好的运行环境，不管到win mac 都能运行

> 解决了运行环境的配置和运维难的问题

## 有什么用 ？

- 环境统一：开发、测试、生产一模一样
- 版本隔离：多版本软件随意装、互不干扰
- 快速部署：一条命令搭建集群、中间件
- 资源占用低： 一台服务器轻松跑几十上百个容器
- 方便集群和管理：后续 Redis 集群、微服务、K8s 全部基于 Docker
