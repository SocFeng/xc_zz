## 一、Go 语言的特点

### 面试问题

Go 语言有哪些核心特点？为什么选择 Go？

### 标准答案

**核心特点：**

1. **编译型语言**：编译为机器码，执行效率高
2. **静态类型**：编译时类型检查，减少运行时错误
3. **原生并发**：goroutine + channel，轻量级并发
4. **垃圾回收**：自动内存管理，降低开发复杂度
5. **简洁语法**：25 个关键字，没有继承、泛型（1.18+ 支持）
6. **快速编译**：大型项目秒级编译
7. **跨平台**：交叉编译方便
8. **内置工具链**：go build、go test、go fmt、go vet

**Go vs 其他语言：**

| 特性 | Go | Java | Python | C++ |
|------|-----|------|--------|-----|
| 编译速度 | 极快 | 慢 | N/A | 慢 |
| 运行速度 | 快 | 快 | 慢 | 极快 |
| 并发模型 | goroutine | 线程 | 协程 | 线程 |
| 内存管理 | GC | GC | GC | 手动 |
| 学习曲线 | 低 | 中 | 低 | 高 |

## 二、Go 的变量声明

### 面试问题

Go 有哪些变量声明方式？它们有什么区别？

### 标准答案

```go
// 1. var 声明（包级别或函数级别）
var name string
var age int = 25
var city = "Beijing"  // 类型推断

// 2. 短声明 :=（只能在函数内部使用）
name := "Alice"
age := 25

// 3. 多变量声明
var (
    a int    = 1
    b string = "hello"
    c bool   = true
)

// 一行多变量
x, y, z := 1, 2, 3

// 4. 常量
const Pi = 3.14159
const (
    StatusOK    = 200
    StatusError = 500
)

// iota 枚举
const (
    Sunday = iota  // 0
    Monday         // 1
    Tuesday        // 2
    Wednesday      // 3
)

// iota 表达式
const (
    _  = iota             // 0
    KB = 1 << (10 * iota) // 1024
    MB                    // 1048576
    GB                    // 1073741824
)
```

**var vs :=：**

```go
// var 可以在包级别使用
var globalVar = "hello"

func main() {
    // := 只能在函数内部使用
    localVar := "world"

    // := 至少要有一个新变量
    x := 1
    x, y := 2, 3  // OK，y 是新变量
    // x := 4     // 编译错误：no new variables
}
```

**零值：**

```go
// Go 中变量声明后会有零值，不会是 nil（指针、slice、map、channel 除外）
var (
    i int       // 0
    f float64   // 0.0
    b bool      // false
    s string    // ""
    p *int      // nil
    sl []int    // nil
    m map[string]int  // nil
    ch chan int  // nil
)
```

## 三、Go 的数据类型

### 面试问题

Go 有哪些基本数据类型？类型转换有什么规则？

### 标准答案

**基本类型：**

```go
// 布尔类型
var b bool = true

// 整数类型
var (
    i8  int8   = 127           // -128 ~ 127
    i16 int16  = 32767         // -32768 ~ 32767
    i32 int32  = 2147483647    // 约 21 亿
    i64 int64  = 9223372036854775807
    i   int    = 42            // 平台相关（32 或 64 位）
    u8  uint8  = 255           // 0 ~ 255（byte 是 uint8 的别名）
    u16 uint16 = 65535
    u32 uint32 = 4294967295
    u64 uint64 = 18446744073709551615
    u   uint   = 42
)

// 浮点类型
var (
    f32 float32 = 3.14
    f64 float64 = 3.141592653589793
)

// 复数类型
var c complex128 = 1 + 2i

// 字符类型（Go 没有 char，用 byte 和 rune）
var (
    b byte = 'A'    // uint8，表示 ASCII
    r rune = '中'   // int32，表示 Unicode 码点
)

// 字符串
var s string = "hello"  // 不可变的字节序列
```

**类型转换：**

```go
// Go 不支持隐式类型转换，必须显式转换
var i int = 42
var f float64 = float64(i)
var u uint = uint(f)

// 数字转字符串
s := strconv.Itoa(42)          // "42"
s := strconv.FormatInt(42, 10) // "42"

// 字符串转数字
i, err := strconv.Atoi("42")           // 42
f, err := strconv.ParseFloat("3.14", 64) // 3.14

// 字符串和 []byte 互转
b := []byte("hello")
s := string(b)

// 字符串和 []rune 互转（处理中文）
r := []rune("你好")
s := string(r)
```

**类型别名 vs 类型定义：**

```go
// 类型别名（完全相同）
type MyInt = int  // MyInt 和 int 是同一个类型

// 类型定义（新类型）
type MyInt int    // MyInt 和 int 是不同的类型

var a int = 1
var b MyInt = 2
// a = b  // 编译错误：类型不同
a = int(b)  // 需要显式转换
```

## 四、Go 的指针

### 面试问题

Go 的指针和 C 的指针有什么区别？什么时候使用指针？

### 标准答案

**基本用法：**

```go
func main() {
    x := 42
    p := &x         // p 是指向 x 的指针
    fmt.Println(p)   // 内存地址
    fmt.Println(*p)  // 42（解引用）

    *p = 100         // 通过指针修改 x
    fmt.Println(x)   // 100
}
```

**Go 指针 vs C 指针：**

| 特性 | Go 指针 | C 指针 |
|------|---------|--------|
| 指针运算 | 不支持 | 支持 |
| 空指针 | nil | NULL |
| 垃圾回收 | 自动 | 手动 |
| 悬空指针 | 不可能 | 可能 |
| 取地址 | `&` | `&` |
| 解引用 | `*` | `*` |

**指针作为函数参数：**

```go
// 值传递（无法修改原变量）
func swap(a, b int) {
    a, b = b, a
}

// 指针传递（可以修改原变量）
func swap(a, b *int) {
    *a, *b = *b, *a
}

func main() {
    x, y := 1, 2
    swap(&x, &y)
    fmt.Println(x, y) // 2, 1
}
```

**什么时候使用指针：**

```go
// 1. 需要修改函数参数
func increment(p *int) {
    *p++
}

// 2. 避免大结构体拷贝
type LargeStruct struct {
    Data [1024 * 1024]byte
}

func process(s *LargeStruct) {  // 传指针，避免拷贝
    // ...
}

// 3. 表示可选值（nil 表示不存在）
func find(name string) *User {
    if user, ok := db[name]; ok {
        return &user
    }
    return nil
}

// 4. 接收者方法需要修改状态
type Counter struct {
    value int
}

func (c *Counter) Increment() {
    c.value++
}
```

**new 和 make 的区别：**

```go
// new：分配内存，返回指针，适用于值类型
p := new(int)     // *int，值为 0
s := new([]int)   // *[]int，slice 本身为 nil

// make：初始化 slice、map、channel，返回初始化后的值（不是指针）
sl := make([]int, 5, 10)    // len=5, cap=10
m := make(map[string]int)   // 空 map，可以使用
ch := make(chan int, 10)     // 带缓冲的 channel
```

## 五、Go 的运算符

### 面试问题

Go 的运算符和其他语言有什么不同？

### 标准答案

```go
// 算术运算符
+ - * / %

// 注意：Go 中 ++ 和 -- 是语句，不是表达式
i++  // OK
// j = i++  // 编译错误：不能作为表达式使用

// 比较运算符
== != < > <= >=

// 逻辑运算符
&& || !

// 位运算符
& | ^ &^ << >>
// &^ 是位清除（AND NOT）
x := 0b1111
y := 0b1100
z := x &^ y  // 0b0011（清除 y 中为 1 的位）

// 取地址和解引用
& *  // 仅用于指针

// 接收操作符（channel）
<-ch    // 从 channel 接收
ch <- v // 向 channel 发送
```
