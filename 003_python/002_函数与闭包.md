## 一、Python 的装饰器

### 面试问题

什么是装饰器？请实现一个带参数的装饰器。装饰器有哪些应用场景？

### 标准答案

**装饰器的本质：** 一个接受函数作为参数并返回新函数的高阶函数，用 `@` 语法糖简化调用。

**基础装饰器：**

```python
def timer(func):
    import time
    def wrapper(*args, **kwargs):
        start = time.time()
        result = func(*args, **kwargs)
        print(f"{func.__name__} 耗时: {time.time() - start:.4f}s")
        return result
    return wrapper

@timer
def slow_function():
    import time
    time.sleep(1)

slow_function()  # slow_function 耗时: 1.0012s
```

**保留原函数信息（functools.wraps）：**

```python
import functools

def timer(func):
    @functools.wraps(func)  # 保留原函数的 __name__、__doc__ 等
    def wrapper(*args, **kwargs):
        # ...
        return func(*args, **kwargs)
    return wrapper
```

**带参数的装饰器（三层嵌套）：**

```python
import functools

def retry(max_retries=3, exceptions=(Exception,)):
    """重试装饰器，支持指定最大重试次数和异常类型"""
    def decorator(func):
        @functools.wraps(func)
        def wrapper(*args, **kwargs):
            for attempt in range(max_retries):
                try:
                    return func(*args, **kwargs)
                except exceptions as e:
                    if attempt == max_retries - 1:
                        raise
                    print(f"第 {attempt + 1} 次失败: {e}, 重试中...")
        return wrapper
    return decorator

@retry(max_retries=3, exceptions=(ConnectionError, TimeoutError))
def fetch_data(url):
    import requests
    return requests.get(url, timeout=5)
```

**类装饰器：**

```python
class Timer:
    def __init__(self, func):
        self.func = func
        functools.update_wrapper(self, func)

    def __call__(self, *args, **kwargs):
        import time
        start = time.time()
        result = self.func(*args, **kwargs)
        print(f"{self.func.__name__} 耗时: {time.time() - start:.4f}s")
        return result

@Timer
def slow_function():
    import time
    time.sleep(1)
```

**多个装饰器叠加：**

```python
@decorator_a
@decorator_b
def func():
    pass
# 等价于：func = decorator_a(decorator_b(func))
# 执行顺序：先 decorator_b，再 decorator_a
```

**常见应用场景：**
- 日志记录
- 权限校验
- 缓存（`functools.lru_cache`）
- 计时器
- 重试机制
- 参数校验
- 限流

## 二、Python 的闭包

### 面试问题

什么是闭包？闭包有什么作用？闭包中的变量绑定有什么注意事项？

### 标准答案

**闭包的定义：** 内部函数引用了外部函数的变量，外部函数返回内部函数。

**三个必要条件：**
1. 有嵌套函数
2. 内部函数引用了外部函数的变量
3. 外部函数返回内部函数

```python
def counter():
    count = 0
    def increment():
        nonlocal count  # 修改外部变量需要 nonlocal
        count += 1
        return count
    return increment

c = counter()
print(c())  # 1
print(c())  # 2
print(c())  # 3
# count 变量被"记住"了，每次调用都在原来的基础上增加
```

**闭包的作用：**

1. **数据隐藏和封装**：不需要类就能保持状态
   ```python
   def make_multiplier(factor):
       def multiply(x):
           return x * factor
       return multiply

   double = make_multiplier(2)
   triple = make_multiplier(3)
   print(double(5))  # 10
   print(triple(5))  # 15
   ```

2. **保持状态**：计数器、缓存、配置
3. **装饰器的基础**：装饰器本质上就是闭包

**变量绑定的坑（延迟绑定）：**

```python
# 错误示例
funcs = []
for i in range(5):
    def func():
        return i  # 这里的 i 是引用，不是值
    funcs.append(func)

for f in funcs:
    print(f())  # 全部输出 4！因为循环结束后 i=4

# 正确写法：用默认参数捕获当前值
funcs = []
for i in range(5):
    def func(i=i):  # 默认参数在定义时求值
        return i
    funcs.append(func)

for f in funcs:
    print(f())  # 0 1 2 3 4

# 或者使用闭包
def make_func(i):
    def func():
        return i
    return func

funcs = [make_func(i) for i in range(5)]
```

**面试加分点：** 闭包中的变量存储在函数对象的 `__closure__` 属性中，可以通过 `func.__closure__[0].cell_contents` 访问。

## 三、Python 的 `lambda` 函数

### 面试问题

`lambda` 函数是什么？和普通函数有什么区别？有哪些使用场景？

### 标准答案

**定义：** 匿名函数，只能包含一个表达式，不能包含语句。

```python
# lambda 参数: 表达式
square = lambda x: x ** 2
print(square(5))  # 25

# 等价于
def square(x):
    return x ** 2
```

**与普通函数的区别：**

| 特性 | lambda | def |
|------|--------|-----|
| 名称 | 匿名 | 有名称 |
| 体 | 单个表达式 | 多条语句 |
| 返回值 | 自动返回表达式结果 | 需要 return |
| 可读性 | 适合简单逻辑 | 适合复杂逻辑 |

**常用场景：**

```python
# 1. sorted() 的 key 参数
students = [("Alice", 85), ("Bob", 92), ("Charlie", 78)]
students.sort(key=lambda s: s[1], reverse=True)
# [('Bob', 92), ('Alice', 85), ('Charlie', 78)]

# 2. map() filter() reduce()
nums = [1, 2, 3, 4, 5]
squares = list(map(lambda x: x**2, nums))       # [1, 4, 9, 16, 25]
evens = list(filter(lambda x: x % 2 == 0, nums)) # [2, 4]

from functools import reduce
total = reduce(lambda a, b: a + b, nums)  # 15

# 3. 字典排序
d = {"b": 2, "a": 1, "c": 3}
sorted_items = sorted(d.items(), key=lambda x: x[1])
# [('a', 1), ('b', 2), ('c', 3)]

# 4. GUI 回调
button.onclick(lambda event: print("clicked"))
```

**面试加分点：** lambda 的名字始终是 `<lambda>`，不利于调试。复杂逻辑应该用 `def`。

## 四、`*args` 和 `**kwargs`

### 面试问题

`*args` 和 `**kwargs` 是什么？参数的传递顺序是什么？

### 标准答案

**`*args`：** 接收任意数量的位置参数，打包成**元组**。

```python
def sum_all(*args):
    print(type(args))  # <class 'tuple'>
    return sum(args)

print(sum_all(1, 2, 3, 4))  # 10
```

**`**kwargs`：** 接收任意数量的关键字参数，打包成**字典**。

```python
def print_info(**kwargs):
    print(type(kwargs))  # <class 'dict'>
    for key, value in kwargs.items():
        print(f"{key}: {value}")

print_info(name="Alice", age=25, city="Beijing")
```

**参数顺序（定义时）：**

```python
def func(a, b, *args, c=10, **kwargs):
    print(f"a={a}, b={b}, args={args}, c={c}, kwargs={kwargs}")

func(1, 2, 3, 4, c=20, x=100, y=200)
# a=1, b=2, args=(3, 4), c=20, kwargs={'x': 100, 'y': 200}
```

顺序：普通参数 → 默认参数 → `*args` → keyword-only 参数 → `**kwargs`

**解包操作：**

```python
# 列表/元组解包为位置参数
def add(a, b, c):
    return a + b + c

args = [1, 2, 3]
print(add(*args))  # 6

# 字典解包为关键字参数
kwargs = {"a": 1, "b": 2, "c": 3}
print(add(**kwargs))  # 6

# 合并字典
d1 = {"a": 1}
d2 = {"b": 2}
d3 = {**d1, **d2}  # {'a': 1, 'b': 2}
```

## 五、Python 的类方法、静态方法、实例方法

### 面试问题

实例方法、类方法、静态方法有什么区别？分别在什么场景下使用？

### 标准答案

```python
class MyClass:
    class_var = "I'm class variable"

    def __init__(self, value):
        self.value = value

    # 实例方法：第一个参数是 self
    def instance_method(self):
        return f"instance: {self.value}, {self.class_var}"

    # 类方法：第一个参数是 cls
    @classmethod
    def class_method(cls):
        return f"class: {cls.class_var}"

    # 静态方法：没有 self 或 cls
    @staticmethod
    def static_method(x, y):
        return x + y
```

| 特性 | 实例方法 | 类方法 | 静态方法 |
|------|---------|--------|---------|
| 装饰器 | 无 | `@classmethod` | `@staticmethod` |
| 第一个参数 | `self`（实例） | `cls`（类） | 无 |
| 访问实例属性 | 可以 | 不可以 | 不可以 |
| 访问类属性 | 可以 | 可以 | 不可以 |
| 调用方式 | 实例.方法() | 类.方法() 或 实例.方法() | 类.方法() 或 实例.方法() |

**类方法常用场景 — 工厂方法：**

```python
class Date:
    def __init__(self, year, month, day):
        self.year = year
        self.month = month
        self.day = day

    @classmethod
    def from_string(cls, date_string):
        year, month, day = map(int, date_string.split('-'))
        return cls(year, month, day)

    @classmethod
    def today(cls):
        import datetime
        d = datetime.date.today()
        return cls(d.year, d.month, d.day)

# 多种方式创建实例
d1 = Date(2024, 1, 1)
d2 = Date.from_string("2024-01-01")
d3 = Date.today()
```

**静态方法常用场景 — 工具函数：**

```python
class MathUtils:
    @staticmethod
    def is_even(n):
        return n % 2 == 0

    @staticmethod
    def factorial(n):
        if n <= 1:
            return 1
        return n * MathUtils.factorial(n - 1)
```

## 六、Python 的 `property` 装饰器

### 面试问题

`property` 是什么？如何实现只读属性？如何使用 setter？

### 标准答案

**作用：** 将方法伪装成属性访问，可以在 getter/setter 中添加逻辑控制。

```python
class Circle:
    def __init__(self, radius):
        self._radius = radius  # 约定：单下划线表示"私有"

    @property
    def radius(self):
        """getter"""
        return self._radius

    @radius.setter
    def radius(self, value):
        """setter"""
        if value < 0:
            raise ValueError("半径不能为负数")
        self._radius = value

    @radius.deleter
    def radius(self):
        """deleter"""
        del self._radius

    @property
    def area(self):
        """只读属性（没有 setter）"""
        import math
        return math.pi * self._radius ** 2

c = Circle(5)
print(c.radius)  # 5  看起来像属性访问，实际调用 getter
print(c.area)    # 78.539...  只读属性
c.radius = 10    # 调用 setter
c.area = 100     # AttributeError: can't set（只读）
```

**传统方式对比：**

```python
# 传统 Java 风格
class Circle:
    def get_radius(self):
        return self._radius

    def set_radius(self, value):
        self._radius = value

c = Circle()
c.set_radius(5)       # 不够优雅
print(c.get_radius())

# Pythonic 风格
c = Circle()
c.radius = 5          # 像属性一样访问
print(c.radius)
```

**使用场景：**
- 数据验证（setter 中检查值）
- 计算属性（只读，动态计算）
- 向后兼容（将属性改为方法时不需要改调用方的代码）

## 七、Python 的函数是一等公民

### 面试问题

Python 中函数是一等公民（First-Class Citizen）是什么意思？

### 标准答案

**含义：** 函数在 Python 中和其他对象（int、str、list）一样，可以被赋值、传递、存储。

**具体表现：**

```python
# 1. 函数可以赋值给变量
def greet(name):
    return f"Hello, {name}"

say_hello = greet
print(say_hello("Alice"))  # Hello, Alice

# 2. 函数可以作为参数传递
def apply(func, value):
    return func(value)

print(apply(len, "hello"))  # 5

# 3. 函数可以作为返回值
def make_adder(n):
    def adder(x):
        return x + n
    return adder

add5 = make_adder(5)
print(add5(10))  # 15

# 4. 函数可以存储在数据结构中
operations = {
    "add": lambda a, b: a + b,
    "sub": lambda a, b: a - b,
    "mul": lambda a, b: a * b,
}
print(operations["add"](1, 2))  # 3

# 5. 函数有属性
print(greet.__name__)  # greet
print(greet.__doc__)   # None
```

**面试加分点：** 这是装饰器、回调函数、策略模式等高级特性的基础。

## 八、Python 的高阶函数

### 面试问题

Python 中有哪些常用的高阶函数？请举例说明。

### 标准答案

**高阶函数：** 接受函数作为参数或返回函数的函数。

**`map()` — 映射：**

```python
nums = [1, 2, 3, 4, 5]
squares = list(map(lambda x: x**2, nums))  # [1, 4, 9, 16, 25]

# 等价于列表推导式
squares = [x**2 for x in nums]
```

**`filter()` — 过滤：**

```python
nums = [1, 2, 3, 4, 5]
evens = list(filter(lambda x: x % 2 == 0, nums))  # [2, 4]

# 等价于
evens = [x for x in nums if x % 2 == 0]
```

**`reduce()` — 累积：**

```python
from functools import reduce

nums = [1, 2, 3, 4, 5]
total = reduce(lambda a, b: a + b, nums)  # 15
# 执行过程：((((1+2)+3)+4)+5)

# 有初始值
total = reduce(lambda a, b: a + b, nums, 100)  # 115
```

**`sorted()` — 排序：**

```python
students = [("Alice", 85), ("Bob", 92), ("Charlie", 78)]
result = sorted(students, key=lambda s: s[1], reverse=True)
# [('Bob', 92), ('Alice', 85), ('Charlie', 78)]
```

**`functools.partial()` — 偏函数：**

```python
from functools import partial

def power(base, exponent):
    return base ** exponent

square = partial(power, exponent=2)
cube = partial(power, exponent=3)

print(square(5))  # 25
print(cube(5))    # 125
```

**面试加分点：** 在大多数情况下，列表推导式比 `map` 和 `filter` 更 Pythonic，可读性更好。
